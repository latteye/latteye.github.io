Openwrt PPTP Howto

这并不是一篇全面的 Howto，仅仅记录了我自己适用、需要的部分。

1. 内核编译

由于 PPTP 需要链接 Windows PPTP 服务器，所以必须要有 mppe 模块，请在编译 openwrt 系统时将 kmod-mppe 模块编译进系统。

2. pppd 配置文件

（1）创建目录

mkdir /etc/ppp/peers

(2) 创建拨号配置文件

vi /etc/ppp/peers/company

pty “pptp ServerIP —nolaunchpppd”

name PPTPusername

remotename PPTP

require-mppe-128

mppe required,stateless ##这一行的写法就和我 Arch Linux 下不同了。

file /etc/ppp/options.pptp

ipparam connect

persist ## 这个应该是断线尝试重播

NOD32 AMON 频繁监控 MSN 导致 System 进程占用CPU资源过高

最近有两位同事的 system 进程在打开 MSN 后占用CPU资源持续在 50%-100%。
之前花过2天时间来看，却没看出啥结果。今天算是了结了这个问题。

在 Documents and Settings{username}\Contacts 下面，会有以你MSN登陆帐号为名的目录。貌似 MSN 对此目录是频繁读写的。
而 NOD32 AMON 服务会即时的监控文件的读取。
这样就导致了 CPU 占用资源非常高。
偏偏又是 System 进程，在检查的时候就要非常小心了。

解决的方法很简单，将对应的目录排除即可。

小鱼搬家记

周末和乖乖帮小鱼换水，忽然发现小鱼缸又有裂痕了（之前乖乖曾用101胶水粘过，暂时起效），于是决定帮小鱼换个房间。去超市买了一个大概10×18×25cm的密封盒，在盒子里铺了一层4-5cm厚的沙子，加入水，放入大苏打中和氯气，又植入两个青青的水草，待水澄清之后小鱼终于搬入了新家。只是，正如现在的我们一样，从老房子入住新居总是有利有弊的：小鱼的新家环境好了，不担心漏水（当然他们的漏水和咱们的概念不一样），但是房价高了（前后两个鱼缸价格差不多，容积新家只有老家的2/3），小鱼的生活空间也受到了限制，正如生活在都市中的我们，高价买个牢笼，还你争我抢的。唉，随便感慨一下！
