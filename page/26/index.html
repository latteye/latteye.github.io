Evig Natt

2007年6月7日

最近很喜欢阴暗的声音。不是金属，不是哥特。

听着悠悠的女声慢慢哼～

午夜，教堂，月亮，破窗，幽灵般缠绵的声音从一个角落传来。你无法知道眼前看到的是不是真实的。昏昏欲睡，浑身无力。

这是 Evig Natt 07年的新专辑，貌似这也是个新乐队。第一张专辑能有这样的质量很不错了，希望不是他们质量最好的一张专辑。很多乐队第一张专辑都会成为一个经典。但是这张绝对不是经典。

听着这样的声音会想起 Elegeion，也会想睡觉。但又睡不着，半梦半醒间的快感～！

Linux 是什么

2007年5月5日

原作者：鸟哥

什么是 Linux ？什么是操作系统？

要了解 Linux 之前，就不能不知道什么是操作系统 ( Operation System, OS )，所以，首先我们来简单的说一说什么是操作系统吧！先来想一想，当我们使用计算机时，屏幕上面显示的咚咚是由哪里来的？嗯！是由显示卡与屏幕显像的；那么你现在可以藉由网络看到这篇文章，则是藉由 Internet 、网络卡、网络线以及所有相关的电子器材与网络器材所完成的一项任务！如果你要看 VCD 呢？那么就需要光驱、光盘、声卡的发声等等的支持。这么说来的话，所以在『工作』的东西都是『硬件』的工作呀！对了！就是这些计算机硬件在工作的。那么硬件怎么工作呢？那就是藉由『操作系统』来达成的啰！这个操作系统就是在沟通你这个使用者跟硬件之间的讯息传递啦！也就是说，没有操作系统，那么你的计算机硬件就只是一堆废铁，什么工作都不能做的！

我的罗曼蒂克迷路史

2007年4月9日

今天早上又起晚了，在去公司的路上突然想到自己迷路的经验实在是太多了…… 写下来消遣消遣吧……

一、小屁孩篇

我的迷路史应该从小学开始说起，再早的我也不记得了。

好像我走路的步伐比一些朋友都要快，所以陈曦小朋友（小学、高中同学）会说：“你总是喜欢走在前面，一直都没变过。”冤枉啊！是你们走的太慢了…………

久而久之我和陈曦走在一起的时候就好像是我在带路，他跟着我走…… 事实上很多时候也的确如此。但问题出在！我不认路！

一天，放学。在距离学校不远的地方有一片老式的房屋。四层楼高度的小新村（那个时候还没有新村的叫法）。我们回家一直是路过那里，从来没有试着穿越过这个小新村。

我说：“走，我们今天从这里面走吧！”
陈曦：“你认识路么？”
我说：“认识！”（这一段有争议，我现在记得我没有说“认识”，但陈曦回忆的时候一口咬定是我说“认识”了，他才会跟我走的。）

于是两个小屁孩走入了从未探索的世界！

（唉……如果当时是个MM的话，我就会说 ‘从此王子和公主过着幸福快乐的生活’ ，可惜啊……）
