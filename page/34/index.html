  • 使用 GShare 快速共享文件

    2007年1月15日

    GShare 是一款局域网内使用的快速共享文件的小工具。当你希望使用 Samba 的时候是不是经常为他的配置而头痛呢？GShare方便的解决了这个问题。本文带你快速上手。

    1. 打开菜单-系统-首选项-文件共享（File Sharing）。

    出现的配置窗口具有如下选项：

    启用文件共享

    启用此选项别人才可通过网络访问你共享的文件。如果你正在使用Gnome 2.14，共享服务将在每次你登录的时候启用。

    允许匿名访问

    如果你希望将您共享的文件完全公开，你就得打开这个选项。而你若不希望匿名访问，则需要填写用户名、密码等资料，而登入的用户必须拥有正确的用户名和密码。

    权限许可

    在此你可以选择是否让登入的用户有权利修改你的文件，更新、删除、添加新文件。

  • lightbird

    2007年1月6日

    Lightbird 是一款为局域网内实现聊天功能的小工具。他也区分客户端和服务端。看起来和 ICQ 的群服务非常类似。
    他使用了mono库，并且理论上支持所有平台（Linux、Windows、MacOS）在 Windows 下你只需要安装.Net FrameWork 和 GTK Sharp 即可运行软件。
    Lightbird 服务器使用了Sqlite3作为用户数据以及聊天内容的存储数据库。
