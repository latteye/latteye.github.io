在ubuntu 6.10 上安装VMware Tools

2006年10月28日

很显然这篇文章是为使用虚拟机的用户准备的，VMware的版本为5.5.2。由于ubuntu 6.10 采用了xorg 7.1版本，所以VMware Tools驱动并不能很好的工作，本文将介绍如何让VMware Tools重新工作在新版本的xorg下。

1.安装编译环境：
sudo apt-get install build-essential linux-headers-uname -r` <!--more--> 2.在VMware的菜单中，点击VM->Install VMware Tools，你将看到有光盘被加载。解压缩VMwareTools*.tar.gz文件。执行命令：
cd ~/Desktop/vmware-tools-distrib
sudo ./vmware-install.pl
`
在这个接下来的过程中，我对所有问题都选择了“是”。唯独不选择：运行vmware-config-tools.pl。

3.给vmware-config-tools.pl打补丁：
wget http://mtnbike.org/vmware/vmware-config-tools-5.5.2-patch-diff.txt sudo chmod u+w /usr/bin/vmware-config-tools.pl sudo patch /usr/bin/vmware-config-tools.pl vmware-tools-config-5.5.2-patch-diff.txt sudo /usr/bin/vmware-config-tools.pl
4.将vmware-toolbox添加为开机时自动运行。

5.重启系统。

如何更新Ubuntu 6.06到Ubuntu 6.10

2006年10月27日

wikitut.org更新了从Ubuntu 6.06 更新到Ubuntu 6.10 的方法。

1.在终端中输入以下命令
gksu "update-manager -c"

2.在出现的更新管理器中选择update（更新）按钮。

3.你会被提示你的sources.list文件将被修改，点击ok就可以继续了。

如何在看电影时关闭屏保？

2006年10月26日

是不是经常在看电影时被定时执行的屏保或者电源管理程序打断呢？crazy___cow写了一个基于python的程序来解决这个问题。

1.下载disablegss.tar.gz文件。
2.解压缩后得到disablegss.py文件，将其复制到/usr/local/bin。
3.创建配置文件：~/.disablegss，这个文件中需要包含你想要禁止屏保和电源管理程序的程序！比如我想在运行firefox的时候停止运行屏保，那么添加：

/usr/bin/firefox
记住，一定要添加程序的源地址。（你可以用ps aux命令查看程序的执行命令位置。）

4.执行disablegss.py命令即可。

如何安装Secure Disk PCMCIA multireader

2006年10月26日

1. 编辑fstab文件。
sudo vi /etc/fstab

2.将以下内容添加至文件中：

/dev/hde1 /media/sd vfat noauto,rw,user,exec, 0 0

3. 新增目录：

sudo mkdir /media/sd

4. 你现在可以在刚才创建的目录中使用你的磁盘了：

mount /media/sd cd /media/sd
