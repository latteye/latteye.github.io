Python 中 while 和 for 的可选 else

2011年1月13日

python 中的 while 和 for 控制结构，最有个性的就是有一个可选的 else 功能。从逻辑上并不是很难接受：

《Python 学习手册》里面解释：只有当 break 的时候才不会走到 else。这解释看的我晕死了。
《可爱的 Python》 里面就说的简单： 当测试条件 True 变成 Flase 的时候，就会走到 else。

逻辑上是明白了，但是具体什么情况下应用，很模糊。
《可爱》里面给的一个例子是：

当你使用 while 寻找你要找的东西时，一旦找到了，给出一个 break。如果遍历完还没有找到，则走到 else ，做一些其他动作。

这的确是实用的例子。
新的东西，完全没感觉，慢慢积累吧。这里只为加深一下印象。
