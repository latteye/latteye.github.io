Acer H340 Home Server

2010年6月14日

Amazon 促销 355 美金买下，要到8月份等老板回家才能帮忙带来。原来想自己拼一个基于 i3 平台的 HomeServer 的，但 Acer 这款机箱实在是太赞了，准备以后就放在电视机柜上。
据说 Newegg 曾经卖过 299 美金，加上税估计在 330 美金左右。而 Amazon 在 CA 是不收税的。所以价格也差不多了。

国内别说 HomeServer 了，一个 4 位的 rack 都要 2000.

4个硬盘位
Atom 230 CPU
2G 内存

跑存储是绝对没问题啦。
