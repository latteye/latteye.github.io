MySQL 报错-解决

2008年8月13日

今天在尝试导入一整个数据库之后，再次进入 mysql shell，一直出现奇怪的问题：

mysql> use test

Database changed

mysql> show tables;

Ignoring query to other database

mysql>

几乎所有指令都报这个错误。

在执行多次： mysql -v -uroot 之后，就好了。
依旧不知道这个问题的起因，以及明确的解决方法。

8月14日更新：

发现，现在如果用 mysql -root 进入 mysql shell，就总是出现同样的问题。
但是若 mysql -uroot 则没有问题。
