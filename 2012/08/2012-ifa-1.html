2012 IFA Day One

IFA 即德国柏林电子消费品展 第一天已经过去了，最受我个人期待的 三星 Galaxy Note 2 也已经发布。但展会并不只有 Note 2 引人注目，众多的电子产品都可能洗牌未来的市场。为了对市场总结，以及自娱自乐，我来写几篇：

平板

华为 — MediaPad 10 FHD：10.1 IPS;1920x1200;1.2G-1.4G CPU;1G-2G RAM;800w像素摄像头;6600mAh;3G(选配) == 2299 - 2999元
Sony — Xperia Tablet S: 10.1 IPS;1280x800;Tegra 3;? CPU;? RAM;500万像素;6000mAH == 2500-3200 元
三星 — ATIV Tab: 10.1 ; 1366x768 ; 1.5G 双核；500万像素; Windows 8;
联想 — IdeaTab A2107: 7 ; 1024x600; 1G A9 双核
联想 — IdeaTab A2109: 9 ; 1280x800; Tegra 3; 1G RAM;300万像素 == 约 1900 元
爱可视 — Arnova Child Pad: 7 ; 1.5G 双核; == 1200 元左右（不含稅？）


从配置上来看，最有诚意的是华为 MediaPad 10 FHD，1200p 的分辨率足以称霸天下。多种可选配置以及 3000 元以内的价格也算实惠。6600mAh 的电池也让人感觉比较靠谱。即使是其最低端的款式（2300元）在联想 IdeaTab A2109 面前也足以秒杀。只是不知道何时才能发售。如果按照华为 Ascend D quad 的跳票次数来看，实在是让人担忧. Ascend D quad 不但错失了最佳利润点，也错失了最佳销售时间。无论如何，华为 MediaPad 10 的出现将 10 寸平板的竞争带入了一个新圈子。


三星 ATIV Tab 实在是毫无特点，让人感觉缺乏诚意，第一款 Windows 产品估计三星自己也这是想摆个造型罢了。不过三星居然在 Nokia 之前发布了 Windows 产品，多少也让 Nokia 脸面丢失。

联想连发三款产品，5寸，7寸，9寸。质量不高，价格相对较低。目标非常明确，攻占低端市场！估计通过其良好的公关也足以吃下大量国内政府采购。


Sony 在进入 Android 时代一直被闷着打。其产品毫无特点，除了靠一点造型来获取少量女性用户的喜爱，实在是看不出其多年前电子产品老大的姿态。

手机

三星 — Galaxy Note II: 5.5;1280x720;1.6G 4核;800万像素
三星 — ATIV S: 4.8;?;1.5G 双核;1G RAM;800万像素;Windows 8
Sony — Xperia T/V/J. T : 4.6;1280x720p;1300万像素. V: 4.3;1280x720p ;LTE;防水;1300万像素. J: 低端款.
HTC — Desire X： 4; 800x480;500万像素


第一天 IFA 手机的展出远比我想像的要少，最耀眼的自然是 Galaxy Note II 的发布了。今天三星股票从被判罚10亿美金狂跌 127亿 美金市值之后反弹 2.9% 靠的就是 Note II.由于 Note I 的大卖（Galaxy Note 全球销量破千万）市场对 Note II 期待甚高。而 Galaxy S III 那在发布时被人狂批外形的手机，在两个月时间即销售 1000万部的傲人业绩也让 Note II 顺理成章的延续了 S III 的外形设计。更大的电池，更窄的边框，体验更好的 S Pen—都成为 Note II 将要热卖的原因。
唯一让人失望的是分辨率没有达到之前预期的 1600x1200 级别。这个问题我觉得要从两点来考虑：

1. 是否有必要？固然 720p 5.5 寸的 PPI 达不到 Retina 那样的效果。但是由于手持设备尺寸的增加，其观看距离有所拉长，即使 PPI 略低，对人观看体验影响也不大。所以 the New iPad 的 PPI 也达不到 iPhone 4 那样的级别。

2. 值得么？1600x1200 不是做不出来，无非是钱而已。而纵观市场上的同类竞争对手 LG Optimus Vu 的 5寸 1024x768 产品只能算是幼儿园级别的，联想 乐Phone K860 纵然也是 5.5寸 720p 级别的产品，但 Note II 2G RAM 笑呵呵的看着联想，再加上3100 mAh 电池，更加薄的机身注定了联想只能是个垫背的。在这样的市场环境下是很难激发 三星 出 1600x1200 的产品的。毕竟 Note II 要是跑到 6000 元档位，就只有看 LG 和联想笑的份了。

三星 的首部 Win 8 手机和平板一样，试水而已。总觉得三星只是出于礼貌…


Sony 终于出了一款 720p 且尺寸大于 4.5 的产品了。4.5 的 720p 那看着真是累啊。刚出一款又把支持 LTE 的 V 放到 4.3 了。Sony 你就这么抠门么！你就这么抠门么！大 0.3-0.4 寸能让你倒闭么！你这做出来是想给人用的东西么。


HTC 说：“我们不做低端产品”，结果 IFA 就出了个 Desire X。好吧，这只是一个补完产品，但是对于 IFA 来说，你是否能拿出个像样的东西来呢，毕竟你也租了像样的展台了，都是要花钱的啊。

更多产品笔电、混合平板/笔电等后续继续分析…
