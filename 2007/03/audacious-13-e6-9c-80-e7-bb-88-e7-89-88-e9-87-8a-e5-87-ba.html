Audacious 1.3 最终版释出

2007年3月3日

从 Audacious 主页获知，1.3最终版本已经释出了。以下为官方内容：

源码：

Audacious: 1.3.0
Audacious-Plugins: 1.3.0
Audacious-Plugins-Ugly: 1.3.0

deb for ubuntu：

Audacious
Audacious-Plugins
Audacious-Plugins-Ugly

Audacious 1.3 is finally released. We would like to thank the community for providing feedback based on the 1.3 RC1 and RC2 releases.

This release’s build-id is: 20070302-4214. If your audacious –version has this build-id, your distribution has pushed 1.3.0 to your computer. This is expected to be common over the next month or so.

Audacious 1.3 introduces some new dependencies which are being shared with other projects in order to promote cross-desktop environment compatibility. These include mcs, which is required to build Audacious 1.3.

Please note that Audacious 1.2 plugins are not binary compatible with 1.3. You will have to recompile (and in some cases will have to update the plugin’s code to the new API) the plugins to make them work. As such, we have bumped the revision of libaudacious to enforce recompiling.

For more details concerning what has changed in Audacious 1.3, visit the 1.3 Release page.

You can discuss the release on our forum, here.
