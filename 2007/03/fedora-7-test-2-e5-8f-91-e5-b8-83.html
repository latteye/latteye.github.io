Fedora 7 Test 2 发布

2007年3月2日


Fedora 7 的第二个测试版本发布了。

此版本添加了许多重要工具的新版本以及一些新技术。而Fedora Core 和 Extras 两个软件库也已经合并。现在你可以从一个源中获得所有的软件。

Fedora 7 进行了3种方式的打包：Desktop / Development Workstation / Server implementation。
