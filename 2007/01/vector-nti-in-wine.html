Vector NTI IN Wine

2007年1月28日

Vector NTI 是一款生物工程软件，在经受了多位前辈的熏陶之后我们应该有这样的思想：“任何科学研究需要的工具Linux都是有的！”可惜的是，Linux似乎并没有生物工程类的软件，没办法！我们只要用Wine模拟了。

misha680 给出了详细的方法：

1.安装 Wine、以及其他必要软件：

sudo aptitude install wine cabextract unzip

2.下载这个脚本，并且给上执行权限。接着运行它，它会为你安装上所有必要的东西。
