aMule CVS 版 071104

2007年11月4日

aMule 2.20 已经蓄势待发了。官方在召集各个语言版本的翻译人员。中文的翻译已经基本完成了。禁不住诱惑，先把 CVS 抓下来用用把。

一、下载源代码

aMule-CVS-20071104.tar.bz2

二、编译

./configure --enable-optimize --enable-amulecmd --enable-amule-daemon --enable-amule-gui --enable-webserver --enable-cas --enable-wxcas --enable-alc --enable-alcc make make install

默认 aMule 将被安装到 /usr/local 下。若希望指定安装位置，使用 –prefix=/xxx/xxx 参数即可。

ps：我向一些前辈学习的程序管理模式是这样的：
程序安装到 /opt/programname-version/ 下。接着在 /usr/share 下做链接 programname 。
所以 aMule 就被安装到 /opt/aMule-CVS-071104 目录下，接着 /usr/share/aMule 指向 /opt/aMule-CVS-071104
最后修改 PATH，设定 PATH=$PATH:/usr/share/aMule/bin 即可。
这样，以后若 aMule 升级，依然安装在 /opt 下，将 /usr/share 下的链接重做即可。

四、aMule web 启动方式

蛮傻的，Web访问还要和GUI分开……
首先 aMule 里面，远程管理 要开启 Web 服务器以及接受外部连接。
然后执行 amuled 命令启动服务。
然后就可以通过 web 访问你的 amule 了。
