Avant Window Navigator 镜像源发布

2007年8月31日

几天前就答应过要搭建 Avant Window Navigator 的镜像源，今天总算是做掉了。

1. 获取密钥

wget http://down.linuxblog.cn/syzygy42/8434D43A.gpg -O- | sudo apt-key add -

2.添加源地址

编辑 /etc/apt/sources.list 文件，加入以下地址：

deb http://down.linuxblog.cn/syzygy42/ feisty avant-window-navigator deb-src http://down.linuxblog.cn/syzygy42/ feisty avant-window-navigator

3. 更新源

apt-get update

4.安装软件

apt-get install avant-window-navigator-bzr libawn-bzr awn-core-applets-bzr

此源每天下午5点会和官方同步一次。
