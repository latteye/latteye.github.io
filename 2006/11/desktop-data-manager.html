Desktop Data Manager

2006年11月14日

Desktop Data Manager是一款全面的剪贴板软件，他可以方便的缓存文字、图片等资料。他还可以快速的进行屏幕截图，可以方便的截取全屏或者是某个窗口。这两个功能基于两个不同的程序。

安装
作者主页上已经提供了deb包和二进制包，其中二进制包解压缩后就可以执行程序了，一般我们不用从源代码编译。
