FVWM-Crystal 3.0.4

FVWM-Crystal是一个轻量级的桌面管理器。它构建在FVWM上。Crystal甚至可以运行在非常老的机器上，这个特点类似于XFCE和Fluxbox。

FVWM-Crystal，原本是作为FVWM的一个主题而出现的。但是今天，它已经进化成了一个更加复杂的项目。它集成了许多有用的工具，列如播放器、任务栏、文件管理器、终端等等。在这种状态下，他完全可以和GNOME或XFCE相比较。而在某些方面胜过Fluxbox。事实上他现在处于他们的中间位置。

为什么要选择Crystal呢？它具有以下几个特点：

1.非常快速！显然Gnome的速度是无法与轻量级的FVWM相比较的。

2.美观。默认安装就可以获得非常棒的视觉效果，你无须花太多时间配置。

3.FVWM-Crystal是可以高度定制的。

安装

我们需要准备以下软件：

  • FVWM - version 2.5.13 or higher
  • Python - for some scripts
  • ImageMagick - import and convert are used in Crystal for thumbnails and desktop screenshots
  • ROX-Filer or Nautilus - for icons on the desktop (additional wallpaper program not needed)
  • XScreenSaver - a screen saver
  • trayer - a systray (notification area)
  • configured sudo
    and optionally gksudo - for running text/graphical programs using root
    privileges and reboot/shutdown computer options in the Exit menu
  • terminal emulator - xterm, aterm, Eterm, mrxvt, urxvt, Eterm, gnome-terminal, konsole or xfce4-terminal
  • GDM (gdmflexiserver) - for shutdown/reboot/new login commands in the Exit menu
  • program for setting a wallpaper - Esetroot, hsetroot or habak
  • music player - Music Player Daemon (+ mpc), X MultiMedia System (+ xmms-shell and XMMSFind), XMMS2, Quod Libet or cdcd
    在终端中运行：
sudo apt-get install fvwm python imagemagick rox-filer xscreensaver trayer gksu aterm habak mpd mpc`</pre>
