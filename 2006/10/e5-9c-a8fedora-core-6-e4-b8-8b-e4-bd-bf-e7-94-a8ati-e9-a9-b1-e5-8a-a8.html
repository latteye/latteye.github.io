在Fedora Core 6下使用ATI驱动

2006年10月31日

本文介绍在Fedora Core 6下驱动显卡的方法。

首先，将你的内核版本从i586升级到i686版本。

yumdownloader kernel-2.6.18-1.2798.fc6.i686 sudo rpm -Uvh --replacefiles --replacepkgs \ kernel-2.6.18-1.2798.fc6.i686.rpm

接着就可以安装显卡驱动了：

su - rpm -Uhv http://rpm.livna.org/livna-release-6.rpm yum install kmod-fglrx xorg-x11-drv-fglrx yum update kmod-fglrx xorg-x11-drv-fglrx service ati-fglrx restart ati-fglrx-config-display enable
